# EFI-Toshiba-L840M

What's working :
1. QE/CI AMD HD 7670M 2GB
2. Backlight slider
3. Fn+F(x) keys
4. Sleep 
5. Bluetooth (Need to boot in windows to trigger bluetooth on mac)
6. USB remapping
7. Audio (Conexant CX20590)
8. Wi-Fi (AR9485 Wireless Network Adapter)
9. Ethernet (AR8151 v2.0 Gigabit Ethernet)
